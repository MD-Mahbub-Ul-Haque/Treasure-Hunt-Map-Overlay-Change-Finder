✕Treasure Hunt — Guide

What it does

Overlay old maps (scanned photos, JPG, PNG, PDF) onto a newer GeoJSON reference, isolate marked features by colour, align everything to real coordinates, and trace features into exportable GeoJSON.

Loading data

  • Reference (GeoJSON) — your baseline "new" map. The view zooms to fit it.
  • ＋ Add maps — add one or more old maps (JPG / PNG / PDF). Each becomes a layer. PDFs import page 1 only.

Layers panel (right)

  • Colour swatch — the colour this map's extracted marks appear in.
  • Eye — show / hide the layer.
  • Name — click to make a layer active. Only the active layer can be moved or edited.
  • ✕ — remove the layer.

Layer settings

  • Fade — layer opacity (your light-table dimmer).
  • Scale / Rotate — resize and spin the map over the ground.
  • Centre here — snap the layer to the middle of the view if it drifts off-screen.
  • Extract markings — Map shows the original; Marks only shows just the extracted colour; Both highlights marks over a dimmed map.
  • Pick a colour, then click a marking to sample it — add several shades. Tolerance loosens or tightens the match. Show marks in sets the output colour.

Tools (bottom-left)

  • Move — drag the active layer.
  • Pick colour — sample marking colours for extraction.
  • Coordinates — live lat/lng at the cursor; click to drop a pin with exact coordinates.
  • Align — the reliable georeferencing tool. Click a landmark on the old map, then the same spot on the new map; repeat for a second landmark far away. Scale, rotation, and position are solved. If rotation comes out flipped, correct it with the Rotate slider.
  • Draw — trace features by hand. Choose Line (river) or Area (lake), click to place points, Undo point / Finish (Enter) / Esc to discard. Export saves everything as GeoJSON; Clear wipes it. Because you draw on the map, the output is already in real coordinates.

Basemaps

Satellite (best for aligning aerial photos), Streets, or None.

Ask Claude

Advisory chat about aligning and extracting. It doesn't move your maps, and connects only inside Claude's own environment — hosted elsewhere it shows an offline note. Everything else works fully in the browser.

Recommended order

Load reference → add old maps → rough-place with Move/Scale/Rotate → Align each precisely → extract markings by colour → trace with Draw → Export to GeoJSON.
